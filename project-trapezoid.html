Project Overview

Trapezoid Technology is revolutionizing trucking logistics with an IoT network of cameras and sensors at facility gates.

Their AI captures and verifies truck and trailer information, solving inefficiencies and trust issues in an industry where many yard management processes are still on paper.

They aim to provide the comprehensive and reliable data needed for a highly automated future, connecting trucking companies, shippers, brokers, and insurers to streamline operations.

The Problem

The initial dashboard design prioritized data display over user experience, resulting in visual clutter and poor usability. Users found it difficult to interpret and interact with key information, which led to low adoption and overall engagement.

Existing dashboard showing cluttered information and poor visual hierarchy

The Solution

Working in agile sprints, we iterated and refined the design across devices to create a fully responsive dashboard. It featured both column and list views for easier data access, along with engaging visual elements such as notifications and a vehicle photo gallery to enrich user interaction.

The redesigned dashboard is now intuitive, visually engaging, and seamlessly organizes large volumes of data across all devices.

Ideation

The Challenge

The client needed to display a large amount of information on a single page, making it difficult to organize content clearly and intuitively.

To tackle this, I drew inspiration from well-organized designs of Fretron, Materio, and Navrrac, crafting a Mid-Fi design that structured data into distinct cards.

I also introduced the idea of a list view that presents information in a table format with collapsible and expandable rows for additional details.

Design Decisions - Informed by Research

Based on user research and real use cases, I designed specific features to address key pain points:

Omni-Search Bar

Universal entry point to search across multiple data types, enhancing efficiency and usability for users needing quick data location.

Gallery View Functionality

Since AI extracted data from captured images, users needed visual verification. The gallery view allowed seamless image review.

Data Accuracy Tags

Designed labeled tags like "Data Matched," "Not Matched," and "Missing Data" to provide immediate visual cues on data reliability.

Edit & Rerun Options

Given AI's potential for incorrect data extraction, I added clear edit and rerun actions with intuitive icons.

Color Palette & Design System

Given the client's identity as a trustworthy and reliable company, I selected blue as the primary color to represent trust and professionalism, with semantic colors for enhanced clarity.

Delivery & Outcomes

Responsive Design Implementation

We created a fully responsive dashboard that works seamlessly across desktop, tablet, and mobile devices. The challenge was displaying large amounts of data on smaller screens while maintaining usability.

Desktop Experience

Full-featured dashboard with both column and list views, advanced filtering, and comprehensive data display.

Tablet Adaptation

Optimized layout focusing on key information first, with collapsible sections for detailed data.

Mobile Optimization

Prioritized essential information with simplified navigation and touch-friendly interactions.

Developer Handoff

I conducted thorough design QA, reviewing all components and elements to verify alignment with the style guide, color palette, and typography. Our team developed a fully connected prototype for seamless user flow presentation.
