1. Push your code to a GitHub repository

3. Select "Deploy from a branch" and choose "main"
4. Your site will be available at `https://[username].github.io/pooja-portfolio`

### Colors
Update CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #000000;
    --secondary-color: #666666;
    --accent-color: #0066cc;
    --background-color: #ffffff;
```

- Replace project descriptions and details
- Modify contact information
- Add your own project images

1. Update the projects grid in `index.html`
2. Add corresponding project detail pages
3. Update navigation if needed
